Module Typing__Aast

Annotated AST (AAST). This AST is used internally by the type inference engine (`Infer.ml`).

type 'a aformula = 'a * 'a aformula_

Generic annotated version of a formula. It is useful to have this generic structure for UID types, and in Infer where we need to store local information along the formula.

| AFVar of Lang.Ast.vname
| ACmp of Lang.Ast.bconstraint

| AAnd of 'a aformula * 'a aformula

type 'a aqformula =

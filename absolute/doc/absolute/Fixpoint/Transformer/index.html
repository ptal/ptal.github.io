Module Fixpoint.Transformer

This module is plugged onto a `Solver` in order to provide additional solving information. It provides statistics by default. By using the notion of "transformers" you can slightly altered the solving process to achieve: 1. Branch and bound (minimization and maximization). 2. Stopping the search under some conditions such as time. 3. Printing some information of your choice. 4. Etc.

Rational: You can add your own transformer inside this module. It is not as extensible as I first wished (for example through the use of functors). One challenge is the management of backtrackable and global states and the dynamic creation of the strategy. If you want to try to make it more extensible, see my trial on combinators(https://github.com/ptal/AbSolute/tree/0332a5b6e66d8138ff86db43294298af6282ec97/src/combinators).

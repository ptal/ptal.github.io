Module Vardom__Vardom_factory.Make

This module aims at extending a vardom with conveniency function derived from `of_bounds`. The variable type (`var_ty`) passed to `of_bounds` depends on the constructor used. For example, `of_ints` calls `of_bounds` with the type `Concrete Int`. A rule of thumb is that we always try to build the vardom with the concrete type of the arguments.

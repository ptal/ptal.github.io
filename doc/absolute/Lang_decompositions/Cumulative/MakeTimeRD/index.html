Similar to `MakeTaskRD.shared_constraints`. Here the decomposition is: `forall(i,t) task_<t.id>_runs_at_<i> <=> at_instant(t.task, i).

val cumulative : rtask list -> D.t -> C.t -> name_factory -> Lang.Ast.formula

Time-resource decomposition of cumulative. Similar to `MakeTaskRD.cumulative`. `horizon` is an upper bound on the latest possible time. The decomposition is `forall(i in 0..horizon) capacity >= sum(t) task_<t.id>_runs_at_<i> * t.resource_usage

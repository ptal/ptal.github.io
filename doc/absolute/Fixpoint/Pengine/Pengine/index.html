val fixpoint : t -> ('a -> task_id -> 'a * bool * event list) -> 'a -> t * 'a

`fixpoint engine f acc` iterates over the scheduled tasks until no task are scheduled anymore. `f acc t` performs the task `t` over `acc` and returns the new `(acc, t_fixpoint, events)` where `acc` is the new accumulator, `t_fixpoint` is true if the task is at fixpoint, and `events` is a list of events to further react on.
